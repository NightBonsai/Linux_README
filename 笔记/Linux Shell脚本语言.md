## Linux Shell脚本语言

### Shell简介：
- 用 C 语言编写的命令语言、程序设计语言
- 用户通过Shell访问操作系统内核
- 用于在命令行界面中执行一系列的命令和程序，以实现自动化任务和系统管理等功能

### 运行.sh文件：

    cd /tmp                     # 进入到脚本所在的目录
    chmod +x ./test.sh   			  # 使得脚本具有可执行权限，此处必须为 ./xxx
    ./test.sh                  	# 执行脚本
